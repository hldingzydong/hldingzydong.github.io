###### Linux Command
1. 解压: tar xvf {file_name}.tar
2. 删除文件: rm {file_name}.tar
3. 删除文件目录: rm -rf {directory_name}
4. ctrl+alt+F1 进入命令行界面(谁叫我出现了循环登陆,没得办法) (For Ubuntun)
5. ctrl+alt+F7 返回到图形登陆界面
6. 移动文件: mv src_file dst
